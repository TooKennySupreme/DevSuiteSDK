

	readme.txt - Test program for LibUSB Midlib transport libraries

- To make: type 'make' at the command line. 


- The test program for the LibUSB Midlib transport is SimpleCapture. This is a Aptina Midlib test program that opens the camera, initializes it, captures 1 or more images and exits. Note: User needs super-user to access the USB devices. (sudo ./SimpleCapture)


- sdat/xsdat file - are located in ../../sensor_data directories.  The sensor_data file contains the target sdat (or xsdat) files.

- apps_data - contains INI files for the different supported sensors.
